# POS Installment Payment - Modal Implementation

## What Changed

Changed the installment setup from inline form to a modal that opens when the checkbox is checked.

## Why Modal is Better

### 1. Cleaner Cart UI
- Cart stays compact and organized
- No expanding forms that take up space
- Better for small screens

### 2. Focused Experience
- Modal provides dedicated space for installment setup
- User focuses on one task at a time
- Less distractions

### 3. Better Mobile Experience
- Modal works better on mobile devices
- Full screen on small devices
- Easier to interact with form fields

## User Flow

### Step 1: Enable Installment
User checks "Enable Installment Payment" checkbox
→ Modal opens automatically

### Step 2: Setup Installments
Modal shows installment setup form:
- Add/remove installments
- Set amounts and due dates
- Real-time validation

### Step 3: Save or Cancel
User can:
- Click "Save Schedule" to confirm
- Click "Cancel" to abort (unchecks the checkbox)
- Click outside modal to close (keeps checkbox if already configured)

### Step 4: View Summary
After saving, cart shows summary:
```
☑ Enable Installment Payment
  3 Installments Configured
  ₱5,000
  [Edit]
```

## Features

### 1. Auto-Open Modal
- Checking the checkbox automatically opens modal
- User immediately sees the setup form
- No extra clicks needed

### 2. Summary Display
After setup, shows:
- Number of installments
- Total amount
- Edit button to reopen modal

### 3. Smart Cancel
- Cancel button unchecks the checkbox
- Clears any configured installments
- Returns to normal state

### 4. Click Outside Behavior
- If installments configured: just closes modal
- If not configured: unchecks checkbox and closes modal
- Prevents accidental data loss

### 5. Validation
- Save button disabled until valid
- Real-time validation feedback
- Clear error messages

## Code Structure

### State Management:
```javascript
const [isStaggered, setIsStaggered] = useState(false);
const [installmentPlan, setInstallmentPlan] = useState(null);
const [showInstallmentModal, setShowInstallmentModal] = useState(false);
```

### Checkbox Handler:
```javascript
onChange={(e) => {
  const checked = e.target.checked;
  setIsStaggered(checked);
  if (checked) {
    setShowInstallmentModal(true); // Auto-open modal
  } else {
    setInstallmentPlan(null); // Clear plan
  }
}}
```

### Modal Close Handler:
```javascript
onClick={() => {
  if (!installmentPlan) {
    setIsStaggered(false); // Uncheck if not configured
  }
  setShowInstallmentModal(false);
}}
```

### Save Handler:
```javascript
onClick={() => {
  if (!installmentPlan) {
    toast.error('Invalid Schedule', 'Please complete the installment setup');
    return;
  }
  setShowInstallmentModal(false);
}}
```
