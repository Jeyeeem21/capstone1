  header: 'Check Date',
  accessor: 'pdo_check_date',
  cell: (row) => {

            month: 'short', day: 'numeric', year: 'numeric' 
          })}
        </span>
      );
    }
    return <span className="text-gray-400">-</span>;
  }
}

WHY ACTION BUTTONS ARE DISABLED:
================================

For PDO payments with status "Pending":
---------------------------------------
- Payment is in two-stage approval process
- Must be approved in PDO Management FIRST
- Action buttons in Payment Transactions are disabled
- Tooltip: "PDO must be approved first in PDO Management tab"

This is CORRECT behavior:
- Prevents skipping the approval workflow
- Ensures proper check verification
- Maintains audit trail

Once approved (status = "Awaiting Payment"):
--------------------------------------------
- ✓ button becomes ENABLED
- Tooltip: "Mark PDO check as paid/cleared"
- Can be marked as paid from Payment Transactions
- Status changes to "Paid"
